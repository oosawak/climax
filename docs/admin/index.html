Climax Chronicle Admin

Sessions / Logs viewer (Function Key protected API)

Connection

GitHub Pages などの静的ホストから使う場合は、Function App のURLを指定します
Functions が AuthLevel.FUNCTION の場合に必要（ブラウザの localStorage に保存します）
未指定なら hostname 相当を想定（運用で固定推奨）

Session

Recent logs から server/session を選ぶと下の入力へ反映します
一覧が空のときは、下の session_id を直接入力してください（"ctm sync" 未実行の可能性があります）
クリックでクリップボードにコピーします（ブラウザから tmux は起動できないので “コピペで起動” 方式）

Logs

過去ログに nlp(intent/provider) を後付けします（topic/limit を使用）
Follow は 5秒毎に再取得します

Output
